NetherGames supports custom in-game emojis that appear directly in chat. To use one, type the emoji's shortcode — surrounded by colons — anywhere in your message. When you send the message, the shortcode is replaced with the corresponding emoji for everyone to see.

<Warning>
Emojis are only visible if you have the NetherGames resource pack installed. When you join the server, click **Download and Join** when prompted to install it.
</Warning>

<Tip>
Use the `/emoji` command in-game to open the emoji help menu at any time.
</Tip>

## Shorthand abbreviations

<Tip>
Some emojis also respond to common text abbreviations. For example, `:)`, `<3`, `:p`, `>:(`, and `:(` are all recognised and converted automatically.
</Tip>

## Preventing emoji conversion

If you want to type a shortcode without converting it to an emoji, prefix it with a backslash (`\`).

```text
\:skull:
```

The text above will appear as-is in chat, without becoming an emoji.
