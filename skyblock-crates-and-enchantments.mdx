Skyblock features a set of special enchantments that go beyond standard Minecraft enchanting. You obtain these enchantments as "enchantment books" by opening crates in the Skyblock lobby. Each crate type gives access to a different tier of enchantment, so the rarer the key you use, the better the enchantment you can receive.

## Crate key types

There are four crate types in Skyblock, each tied to a different source:

| Key type | How to obtain |
|----------|---------------|
| Vote     | Earned by voting at [ngmc.co/vote](https://ngmc.co/vote) |
| Common   | Earned by mining |
| Rare     | Earned by mining |
| Mythic   | Earned by mining |

## Special enchantments

The following enchantments are exclusive to Skyblock and can only be obtained through crates.

<AccordionGroup>
  <Accordion title="Glow — Common">
    **Slot: Helmet**

    Grants night vision to the player wearing the enchanted helmet, active at all times while worn.
  </Accordion>

  <Accordion title="Miner (Levels 1 & 2) — Rare">
    **Slot: Pickaxe**

    Applies a Haste effect while you hold the pickaxe. Level 1 grants Haste I and Level 2 grants Haste II.
  </Accordion>

  <Accordion title="Boom — Rare">
    **Slot: Pickaxe**

    Each block you break has a chance to destroy all blocks in a 3×3×3 area centred on the block you are mining.
  </Accordion>

  <Accordion title="Accelerate — Rare">
    **Slot: Boots**

    Grants Speed II to the player wearing the enchanted boots, active at all times while worn.
  </Accordion>

  <Accordion title="Rabbit — Rare">
    **Slot: Boots**

    Grants Jump Boost III to the player wearing the enchanted boots, active at all times while worn.
  </Accordion>

  <Accordion title="Poison — Rare">
    **Slot: Sword**

    Each hit you land on another player with the enchanted sword has a chance to inflict Poison I on them.
  </Accordion>

  <Accordion title="Triple Shot — Mythical">
    **Slot: Bow**

    Each shot fires between one and three arrows instead of one, while only consuming a single arrow from your inventory.
  </Accordion>

  <Accordion title="Guardian Angel — Mythical">
    **Slot: Armour**

    When your health drops below three hearts in combat, it is immediately restored. This effect has a 30-minute cooldown per use. Equipping multiple items with Guardian Angel does not stack the effect.
  </Accordion>
</AccordionGroup>

<Note>
  Enchantment books are obtained randomly from crates based on the loot table for each key type. Higher-rarity keys give access to higher-rarity enchantments. See the [Skyblock crate keys](/skyblock-crate-keys) page for full loot table details.
</Note>
