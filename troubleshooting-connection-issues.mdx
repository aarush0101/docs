---
title: "Troubleshooting Connection Issues on NetherGames"
description: "Fix common problems like high ping, 'You are already online,' 'Invalid IP address,' and getting stuck on 'Locating server' when joining NetherGames."
---

If you're having trouble connecting to NetherGames — including high ping, being stuck on "Locating server," or seeing errors like "You are already online" or "Invalid IP address" — try the steps below in order.

<Note>
  If you're seeing a **"This username is already registered"** message, that's a separate issue. See the [This Username Is Already Registered](/this-username-is-already-registered) article for help.
</Note>

<AccordionGroup>
  <Accordion title="Restart Minecraft and your device">
    Close Minecraft completely and restart your device. This clears temporary network state and resolves many transient connection errors. Once your device has restarted, relaunch Minecraft and try connecting again.
  </Accordion>

  <Accordion title="Disable any VPN or proxy service">
    If you're using a VPN or proxy, disable it before attempting to connect. These services can interfere with the connection to NetherGames and are a common cause of "Invalid IP address" and similar errors.
  </Accordion>

  <Accordion title="Try a different skin">
    Some skins can cause connection issues and may be blocked. Switch to a different skin and try joining again to rule this out as the cause.
  </Accordion>

  <Accordion title="Try a regional server address">
    Instead of the main address, try connecting directly to a regional server on port **19132**:

    | Region        | Address                  |
    | ------------- | ------------------------ |
    | Asia-Pacific  | `ap.nethergames.org`     |
    | Europe        | `eu.nethergames.org`     |
    | United States | `us.nethergames.org`     |

    Connecting to the region closest to you may reduce ping and resolve routing issues.
  </Accordion>

  <Accordion title="Switch to a different internet connection">
    Try connecting using mobile data instead of Wi-Fi (or vice versa). This helps determine whether the problem is specific to your network. If the issue only occurs on one connection, your ISP or router may be blocking or limiting traffic to the server.
  </Accordion>
</AccordionGroup>

<Warning>
  If none of the steps above resolve your issue, our support team can help. Select **Chat with us** at [ngmc.co/request](https://ngmc.co/request) and describe the problem, including any error messages you're seeing.
</Warning>
