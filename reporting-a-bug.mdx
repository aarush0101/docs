---
title: "How to Report a Bug on NetherGames"
description: "Report gameplay bugs to the NetherGames team via the Discord /bug command, which creates a tracked thread for discussion and developer review."
---

Found something broken in the game? You can report it directly to the NetherGames development team through the official Discord server. Bug reports submitted this way are tracked in a dedicated forum channel where other players can weigh in and developers can triage the issue.

## How to submit a bug report

<Steps>
  <Step title="Join the NetherGames Discord server">
    If you haven't already, join the server at [ngmc.co/d](https://ngmc.co/d) and complete the membership verification process.
  </Step>
  <Step title="Go to the #bots channel">
    Once verified, navigate to the **#bots** channel in the server.
  </Step>
  <Step title="Run the /bug command">
    Type `/bug` and follow the prompts to describe the issue you encountered.
  </Step>
</Steps>

After you submit, a new thread is automatically created in the **#bugs** forum channel. You can continue the discussion there, share additional details, and track the status of your report as the development team reviews it.

<Tip>
Include as much detail as possible when filling out the bug report — what you were doing, which game or mode was affected, and any steps that reproduce the issue consistently. This helps developers investigate and fix the problem faster.
</Tip>
